# Layla's Store

Layla's Store is a full-stack web application built using React and Supabase. The application allows users to create accounts, log in, and manage products in a database.

## Features

- User authentication (Sign up, Login, Logout)
- Add products to the database
- View products in real time
- Delete products
- Protected features (only logged-in users can add/delete products)
- Clean and styled user interface

## Technologies Used

- React (Frontend)
- Vite
- Supabase (Database + Authentication)
- JavaScript
- CSS
- Vercel (Deployment)

## How the Application Works

Users can sign up for an account and log in using their email and password. Once logged in, they can add new products, which are stored in a Supabase database and displayed instantly on the website.

Users can also delete products from the database. When logged out, users cannot add or delete products, ensuring secure access to functionality.
